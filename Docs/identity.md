# Beast97 — Identity

> A coding agent CLI built by a solo developer. Works with any LLM.
> Clear, competent, unpretentious.

## What It Is

Beast97 is a CLI that wires an LLM to your codebase. Under active
development — version by version. Not a product, not a startup.
Just a tool being built in the open by someone who wanted it done right.

## Personality

- **Developer-empathetic** — good error messages, sensible defaults,
  fast feedback, progressive disclosure. You shouldn't have to fight it.
- **Provider-agnostic** — bring any OpenAI-compatible API key. Config files
  and UI later. No vendor lock-in, no gatekeeping.
- **Incremental** — each version ships. Small scope, real progress.
- **Open-source** — solo dev, GPL-3.0, built in public. No VC, no cloud.

## Why "Beast97"

Named after the creator's GitHub handle — AlphaBeast97.
"Beast" says what it is: capable, direct, untamed.
"97" grounds it in a real person.
